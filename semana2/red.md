¿Que es una IP?

Una dirección IP es un identificador númerico para dispositivo.

¿Que es un localhost?

Es un nombre especial que se refiere a tu propia computadora, permitiendo acceder a servicios.
(como servidores web o bases de datos) instalados en ella de forma local , sin usar internet, usando la direccion IP de bucle invertido (loopback) 127.0.0.1.

¿Que es un puerto?

Es un numero (ej: 80, 443, 22) que especifica un canal de comunicación dentro de una ip, para que diferentes aplicaciones puedan coexistir.

Puerto 80: Es para HTTP
Puerto 443: Es para el tráfico de web HTTPS
Puerto 22: Para acceso remoto seguro SSH
Puerto 3306: Es para mySQL

Comandos

ip: para saber mi ip
ping: para saber que hay conectividad
ss: sirve para investigar y mostrar estadisticas detalladas sobre sockets y conexiones de red.
